三人行必有我师

发表于 2019-11-02

看到公众号李三科的一篇文章，一位名叫赵飞的小伙分享自己的成长经历，有明确的目标，为自己的目标建立能力模型，每天学习培养目标能力。很赞。

在微博上看到陈冲写许知远拍摄《十三邀》时，和他的交往互动，字里行间流出宁静、细腻，很佩服她的文采。

后来我们的对话，也经常从书开始。我在泰国拍戏的时候，正逢雨季，雨水蒙住了窗外面的昭皮耶河，把我像蝉蛹一样裹在屋里阅读、听音乐，与世隔绝。接连不断的倾盆大雨让我想起毛姆的精湛短篇《雨》，就跟许知远聊起了毛姆在东南亚和太平洋岛屿写的一系列悲剧，都是关于亚寒带的欧洲人到了融化与腐蚀一切的热带后的生活。也许我俩都属于那种有古典情怀的人吧，从毛姆的作品，我们聊到悲剧的价值。
